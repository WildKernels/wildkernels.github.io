Why Choose Wild Kernels?

KernelSU Next

A kernel-based root solution for Android devices with kernel-based su and root access management, module system based on Magic Mount and OverlayFS, and App Profile to limit root privileges per app. Supports Android kernels from 4.4 up to 6.6 with architectures: arm64-v8a, armeabi-v7a and x86_64.

ඞ

SUSFS Support

An addon root hiding kernel patches and userspace module for KernelSU. The userspace tool ksu_susfs, as well as the ksu module, require a susfs patched kernel to work.
